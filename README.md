# Streaming Fraud Detection Pipeline

## Overview
Detect anomalies in transaction streams using Spark Streaming and Kafka.

## Technologies 
- Spark Streaming 2.4
- Apache Kafka 2.3
- Redis 5.0
- Python 3.9+

## Features
- Real-time fraud detection
- Pattern matching
- Velocity checks
- Geo-location analysis
- Alert generation
- Model scoring
